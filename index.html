Udacity Arcade Game Project

I was provided with visual assets and a game loop engine; using these tools I added number of entities to the game including the player characters and enemies to recreate the classic arcade game Frogger.

Udacity Neighborhood Map Project

I developed a single-page application featuring a map of my neighborhood. I added additional functionality to this application, including: map markers to identify popular locations, a search function to easily discover these locations, and a listview to support simple browsing of all locations. I researched and implemented third-party APIs that provide additional information about each of these locations (such as StreetView images, Wikipedia articles, Yelp reviews, etc).
